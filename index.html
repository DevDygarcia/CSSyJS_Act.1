Actividad 1 - Guía Básica de CSS

By Laura García (DevDygarcia)

1. Definición de CSS

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación de un documento escrito en HTML o XML. Permite controlar el diseño y la apariencia de las páginas web.

2. Funcionamiento Básico de CSS

CSS funciona aplicando estilos a los elementos HTML. Estos estilos pueden incluir colores, fuentes, tamaños, márgenes, y más. Los estilos se aplican en cascada, lo que significa que las reglas pueden heredarse y sobrescribirse.

3. Incluir CSS en un archivo HTML

Hay tres formas de incluir CSS en un archivo HTML:

  • En línea: usando el atributo style en los elementos HTML.
  • Interno: dentro de una etiqueta <style> en el encabezado del documento HTML.
  • Externo: enlazando a un archivo .css separado usando la etiqueta <link>.

4. Definición de una regla CSS

Una regla CSS está compuesta por un selector y un bloque de declaración.

5. Elemento de una regla CSS

  • Regla: Conjunto de estilos aplicados a uno o más elementos.
  • Selector: Indica a qué elementos HTML se aplicarán los estilos.
  • Bloque de declaración: Contiene una o más declaraciones CSS.
  • Propiedad: Aspecto del elemento que se quiere estilizar (por ejemplo, color, tamaño).
  • Valor: Especifica cómo se debe estilizar la propiedad (por ejemplo, rojo, 16px).

6. Medios CSS

Los medios CSS permiten aplicar estilos diferentes según el dispositivo o el medio (pantalla, impresión, etc.). Se utilizan con la regla @media.

7. Comentarios CSS

Los comentarios en CSS se escriben entre /* y */. No se muestran en el navegador y se utilizan para dejar notas en el código.

8. Sintaxis de la definición de las propiedades CSS

La sintaxis básica de una propiedad CSS es:

selector {
    propiedad: valor;
}

Ejemplo:

p {
    color: blue;
    font-size: 16px;
}
